## GitHub Integration

Connect AI Drone to GitHub to automate documentation updates from repository events. You push changes to your repo, and AI Drone syncs them automatically.

<Steps>
  <Step title="Create App" icon="settings">
    Go to GitHub Settings > Developer settings > GitHub Apps. Create a new app with `ai-drone` as the name.
  </Step>
  <Step title="Add Permissions" icon="shield">
    Grant `contents:write` and `pull_requests:read` permissions.
  </Step>
  <Step title="Install and Webhook" icon="link">
    Install on your repo and set webhook URL to `https://api.example.com/v1/webhooks/github`.
  </Step>
</Steps>

<Callout kind="tip">
  Test the webhook with a sample push event to verify delivery.
</Callout>

## Slack Notifications

Send AI Drone events to Slack channels for team alerts. Configure incoming webhooks in Slack and point them to AI Drone's endpoint.

<Tabs>
  <Tab title="Setup" icon="settings">
    In Slack, create an incoming webhook at `https://api.slack.com/apps`. Note the webhook URL.
  </Tab>
  <Tab title="Configure AI Drone" icon="code">
````javascript
// In your AI Drone dashboard, add Slack integration
const slackConfig = {
  url: "https://hooks.slack.com/services/YOUR_SLACK_WEBHOOK",
  channel: "#docs-updates"
};
````
  </Tab>
</Tabs>

## Webhook Setup for Real-Time Updates

Set up webhooks to receive events like doc publishes or approvals. AI Drone sends `POST` requests to your endpoint.

<Request tabs="Payload,Headers" show-lines="true">
````javascript
// Event: document.published
{
  "event": "document.published",
  "doc_id": "doc_123abc",
  "timestamp": 1699123456,
  "user_id": "user_xyz"
}
````
````javascript
// Headers
Authorization: Bearer YOUR_API_KEY
X-AI-Drone-Signature: sha256=abc123...
Content-Type: application/json
````
</Request>

Follow these steps to verify incoming webhooks:

<Steps>
  <Step title="Expose Endpoint" icon="globe">
    Use ngrok or a public server for `https://your-webhook-url.com/webhook`.
  </Step>
  <Step title="Configure in Dashboard" icon="dashboard">
    Add your URL in AI Drone settings under Integrations > Webhooks.
  </Step>
  <Step title="Test Event" icon="zap">
    Trigger a test publish and check your logs.
  </Step>
</Steps>

## API Access for Custom Connections

Build custom integrations using AI Drone's REST API at `https://api.example.com/v1`. Authenticate with a bearer token.

<ParamField header="Authorization" param-type="string" required="true">
  Bearer `YOUR_API_KEY`.
</ParamField>

<ParamField path="integrations/{id}" param-type="string" required="false">
  Integration ID for updates.
</ParamField>

<CodeGroup tabs="JavaScript,Python">
````javascript
const response = await fetch('https://api.example.com/v1/integrations', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    type: 'slack',
    config: { url: 'https://hooks.slack.com/...' }
  })
});
````
````python
import requests
response = requests.post(
    'https://api.example.com/v1/integrations',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={'type': 'slack', 'config': {'url': 'https://hooks.slack.com/...'}
    }
)
````
</CodeGroup>

## Syncing with Project Management Tools

Integrate with Jira, Trello, or Asana to link docs to tasks. Use webhooks or polling APIs to keep everything in sync.

For Jira:

```javascript
// Poll for updates
fetch('https://api.example.com/v1/sync/jira?project=PROJ-123');
```

This setup streamlines your workflow, keeping teams aligned across tools. Start with GitHub or Slack for quick wins.
